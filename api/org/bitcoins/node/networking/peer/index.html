  6.  class PeerMessageSender extends BitcoinSLogger

    Created by chris on 6/7/16.

    Created by chris on 6/7/16. This actor is the middle man between our Client and higher level actors such as org.bitcoins.node.networking.BlockActor. When it receives a message, it tells Client to create connection to a peer, then it exchanges VersionMessage, VerAckMessage and org.bitcoins.node.messages.PingMessage/PongMessage message with our peer on the network. When the Client finally responds to the NetworkMessage we originally sent it sends that NetworkMessage back to the actor that requested it.
